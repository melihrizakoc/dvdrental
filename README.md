# 🎬 DVD Rental System

Full-stack uygulama: .NET Core 9 (Backend) + Angular 19 (Frontend)

## 📦 Proje Yapısı

- `dvdrental.api/` → .NET Core Web API
- `dvdrental.client/` → Angular frontend
- `dvdrental.sln` → Çözüm dosyası

## 🚀 Başlatma Adımları

### Backend (.NET Core)
```bash
cd dvdrental.api
dotnet restore
dotnet run
```

### Frontend (Angular)
```bash
cd dvdrental.client
npm install
ng serve
```
